# Custom Bot Maganer для CSGO
Данный плагин переименует ботов автоматически

Изменение ника бота происходит через команду cbm_botname "ник" (обязательно кавычки!!)

Так же можно добавить клантег боту через команду cbm_bottag "клантег" (обязательно кавычки!!)

После добавление имени и клантега добавляете бота через bot_add, и он будет иметь указанные вами ник и клантег
